# Habit Hero 

Habit Hero is a productivity app designed to help users track habits, build routines,
view analytics such as streaks and success rate, and stay consistent.

---

## Features
- Create habits (daily, weekly)
- Categorize habits (fitness, productivity, learning)
- Track progress using check-ins or notes
- View analytics: streaks, best days, completion %
- Modern UI + Fast API backend

---

## Tech Stack

Backend : Python, FastAPI, SQLModel, SQLite 

Frontend : React + Vite

Tools : Uvicorn, Axios, VSCode

## Getting Started

Follow the steps below to set up and run the project.

---

### Backend Setup (FastAPI)
cd backend

pip install -r requirements.txt
1. Create and activate virtual environment:

python -m venv venv
venv\Scripts\activate # Windows

pip install -r requirements.txt

uvicorn main:app --reload

API will be available at:  
http://127.0.0.1:8000

## Frontend

cd frontend

npm install

npm run dev

UI will run at:  
http://localhost:5173 (or similar port shown in terminal)
